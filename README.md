## Tools / Products Section

This is the main showcase area where all digital tools are displayed as beautiful cards.
Each tool card contains:

### Tool Icon

### Tool Name

### Short Description

### Price + Period (e.g., $10 monthly)

### Tag (popular, best seller, new)

### TagType (category like Productivity, Design, AI)

### List of key features

### "View Details" or "Buy Now" button

Currently, the platform showcases 9 premium tools including Notion, Figma, ChatGPT Plus, Canva Pro, Midjourney, Framer, Zapier, Grammarly, and Loom.

## Pricing Section

The pricing section displays the tools in a clean grid or list format so users can easily compare prices.
It shows:
Monthly pricing for each tool
Popular / Best Seller / New tags to help users decide quickly

Users can filter or sort tools by category (Productivity, Design, AI, etc.) or by popularity.
